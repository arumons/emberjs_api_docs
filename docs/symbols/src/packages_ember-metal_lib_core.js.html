 18   You can also use the shorthand "Em" instead of "Ember".

 65 /**
 66   @static
 67   @type Boolean
 68   @default true
 69   @constant
 70 
 71   Determines whether Ember should enhances some built-in object
 72   prototypes to provide a more friendly API.  If enabled, a few methods
 73   will be added to Function, String, and Array.  Object.prototype will not be
 74   enhanced, which is the one that causes most troubles for people.
 75 
 76   In general we recommend leaving this option set to true since it rarely
 77   conflicts with other code.  If you need to turn it off however, you can
 78   define an ENV.EXTEND_PROTOTYPES config to disable it.
